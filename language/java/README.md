## Java 개발 환경 설정

### Java 설치

> window에서 Java 설치하기

#### 버전 확인

* Java 버전 확인
  * 입력

    ```bash
    java -version
    ```

  * 출력

    ```bash
    java version "15.0.1" 2020-10-20
    Java(TM) SE Runtime Environment (build 15.0.1+9-18)
    Java HotSpot(TM) 64-Bit Server VM (build 15.0.1+9-18, mixed mode, sharing)
    ```

* Javac 버전 확인

  * 입력

    ```bash
    javac -version
    ```

  * 출력

    ```bash
    javac 15.0.1
    ```
