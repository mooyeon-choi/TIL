## Infinite Scroll

* [바로가기](./infinite-scroll)

```
많은 데이터를 받아와야 할 때, 화면에 보이지 않는 부분까지 한번에 받아온다면📱
초기 구동 시간🕒이 매우 오래 걸릴 것이다.
이를 위해 한번에 볼 수 있는 만큼만 먼저 받아오고 스크롤을 끝까지 내리면
또 필요한 만큼 받아와서 표시하면 훨씬 효율적으로 데이터를 받아 올 수 있다🙌
```
