## 운영 체제란 무엇인가

* [바로가기](./what_is_os)

* 운영 체제란(Operating System, OS)란?

	* 운영 체제의 분류

		* 동시 작업 가능 여부

			* 단일 작업(Single tasking)

			* 다중 작업(Multi tasking)

		* 사용자의 수

			* 단일 사용자(Single user)

			* 다중 사용자(Multi user)

		* 처리방식

			* 일괄 처리(Batch processing)

			* 시분할(Time sharing)

			* 실시간(Realtime OS)

			* 실시간 시스템의 개념 확장

	* 혼동하기 쉬운 Multi-

	* 운영 체제의 예

		* 유닉스(UNIX)

		* DOS(Disk Operating System)

		* MS Windows

		* Handheld device를 위한 OS

	* 운영체제의 구조

		* CPU 스케줄링

		* 메모리 관리

		* 파일 관리

		* 입출력 관리

		* 프로세스 관리

		* 그 외
