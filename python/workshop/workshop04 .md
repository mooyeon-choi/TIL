* 양의 정수 x를 입력 받아 제곱근의 근사값을 반환하는 함수를 작성하세요.
* sqrt() 사용 금지


```python
def my_sqrt(n):
    a, b = 1, n
    result = 1
    while abs(result**2 - n) >= 1e-10:
            result = (a+b)/2
            if result**2 < n:
                a = result
            else:
                b = result
            print(result)
    return result
```
